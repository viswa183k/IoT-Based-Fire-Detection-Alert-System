# IoT_Fire_Alert_System

## Description
An IoT-based fire detection system using ESP8266, flame sensor, Blynk IoT app, buzzer, and LED. Sends instant alerts to your smartphone when fire is detected.

## Hardware Required
- ESP8266 NodeMCU
- Flame Sensor Module
- Buzzer
- LED
- Jumper Wires

## Software Required
- Arduino IDE
- Blynk IoT App

## Usage
1. Connect the flame sensor, LED, and buzzer as per the code.
2. Update WiFi credentials and Blynk auth token.
3. Upload `IoT_Fire_Alert_System.ino`.
4. Get real-time alerts on your mobile when fire is detected.
